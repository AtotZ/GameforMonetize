# GameforMonetize

This repo contains a live Pythonista/iPhone workflow for parsing Uber offer cards, scoring them in real time, and logging traffic beacons from iOS shortcuts.

## Read This First

If you are continuing this project in a future chat or handing it to another engineer, start in this order:

1. `IOS_APP_HANDOFF.md`
2. `IOS_APP_IMPLEMENTATION_PLAN.md`
3. `PYTHONISTA_PORTFOLIO.md`

These files explain:

- the current live Pythonista system
- the parent OnisAI architecture this work inherits from
- the intended native iOS app direction
- which ideas are proven, shadow-mode, or still aspirational
